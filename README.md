# 平衡铜绘制 (Balance Copper)

嘉立创EDA专业版扩展 — 自动在PCB空白区域填充非功能性平衡铜图案，平衡各层铜密度，提升电镀均匀性和制造良率。

## 功能特性

### 支持的图案类型

| 图案 | 说明 | ���数 |
|------|------|------|
| 圆形 | 圆点填充 | 直径 |
| 正方形 | 正方形填充 | 尺寸 |
| 矩形 | 矩形填充 | 宽度、高度 |
| 菱形 | 菱形填充（以宽高为对角线） | 水平对角线、垂直对角线 |
| 长圆形 | 两端半圆的长圆形 | 宽度、高度 |
| 三角形 | 等腰三角形 | 底边、高度 |
| 五边形 | 正五边形 | 边长 |
| 六边形 | 正六边形 | 边长 |
| 梯形 | 等腰梯形（顶边=底边/2） | 底边、高度 |

### 核心功能

- **自动避让**：精确避让走线（胶囊形）、焊盘（实际形状+旋转）、过孔、铜皮、已有填充、挖槽区域
- **封装内避让**：解析封装文件，避让封装内焊盘（含旋转）和 MULTI 层挖槽区域（圆形/矩形/任意多边形）
- **DRC间距**：自动读取DRC规则矩阵，按图元类型分别应用安全间距（走线/焊盘/铜皮/板边）
- **交叉分布**：可选隔行交错排列；邻层交错使相邻层图案错开半个步长
- **旋转角度**：支持任意角度旋转图案
- **操作范围**：仅当前层 / 所有信号层 / 所有阻焊层 / 信号层+阻焊层
- **区域生成**：在画布上框选特定区域进行填充
- **自动DRC**：生成完成后可自动运行DRC检查
- **输入框计算**：参���输入框支持四则运算（如输入 `5+3` 自动计算为 `8`）
- **单位自适应**：自动识别当前EDA单位（mil/mm/inch）并转换
- **中英双语**：自动跟随EDA语言设置

### 间距说明

间距参数为图案边缘之间的 **间隙（gap）**，而非中心距。实际网格步长 = 图案尺寸 + 间距。支持分别设置水平和垂直间距。

## 安装

1. 在 [Releases](../../releases) 页面下载最新的 `.eext` 文件
2. 打开嘉立创EDA专业版 → 扩展 → 管理扩展 → 安装本地扩展
3. 选择下载的 `.eext` 文件

## 使用方法

1. 打开PCB设计文档
2. 菜单栏 → 扩展 → 平衡铜工具 → 平衡铜绘制
3. 选择图案类型和参数
4. 点击"一键生成"或"按点击区域生成"

## 项目结构

```
src/
  index.ts                  — 入口：菜单注册、iframe管理、命令轮询
  types.ts                  — 配置接口与枚举定义
  core/
    constants.ts            — 层ID、单位转换、默认值
    polygonUtils.ts         — 几何工具（多边形解析、BBox、距离计算）
    boardOutline.ts         — 板框提取（折线/填充/区域，含圆角矩形和槽孔）
    obstacleCollector.ts    — 收集目标层所有障碍物（填充/铜皮/走线/焊盘/过孔/挖槽区域）
    clearanceEngine.ts      — 网格化空白区域检测（空间索引+安全间距+近似补偿）
    patternGenerator.ts     — 9种图案生成（批量创建Fill图元）
    footprintParser.ts      — 封装文件解析（提取焊盘和挖槽区域）
iframe/
  index.html                — 完整UI面板（内联CSS+JS，中英双语）
extension.json              — 扩展元数据和菜单
locales/                    — i18n翻译文件
```

## 开发

```shell
git clone --depth=1 https://github.com/easyeda/eext-balance-copper.git
cd eext-balance-copper
npm install
npm run build
```

生成的扩展包位于 `./build/dist/` 目录。
